Manually edit array item by index

Usage: Vue.set(ARRAY, INDEX, NEW_VALUE)

e.g., Vue.set(app.todoList, 0, { id: 'a', title: 'Buy 2 books' })

Usage: Vue.set(OBJECT, KEY, VALUE)

e.g., Vue.set(app.object, 'gender', 'man')
